## 3. 新接口说明（后端）

### 资金流向记录
- `GET /api/statistics/flow-records/?year=YYYY&month=M`
- `POST /api/statistics/flow-records/`

POST body 示例：
```json
{
  "source_name": "工资",
  "source_type": "income",
  "target_name": "招商银行卡",
  "target_type": "account",
  "amount": "18000.00",
  "flow_date": "2026-03-28",
  "description": "月薪入账"
}
```

### 资金流向图
- `GET /api/statistics/flow-graph/?year=YYYY&month=M`
- 返回：`nodes + links + summary`

### 热力图
- `GET /api/statistics/heatmap/?year=YYYY&mode=expense|income|net`
- 返回：`[{ date, value }]`

### 资产网络
- `GET /api/statistics/asset-network/`
- 返回：`nodes + links`
